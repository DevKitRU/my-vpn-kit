# iOS — Shadowrocket

Настройка VPN на iPhone/iPad через Shadowrocket.

> Одного скрипта для iOS нет — в Apple SDK нельзя программно настраивать VPN из скрипта. Инструкция пошаговая, 3-5 минут.

## Что потребуется

1. **Shadowrocket** — $2.99 в App Store (купить нужно с аккаунтом США/Турции — в РФ App Store приложения нет). Ссылка: [apps.apple.com/us/app/shadowrocket](https://apps.apple.com/us/app/shadowrocket/id932747118)
2. **VPS с 3x-ui** и **VLESS-ссылка** от него
3. **iOS 14+** (проверено на 17 Pro)

## Импорт VLESS в Shadowrocket

**Способ 1 — через буфер:**
1. Скопируй свою VLESS-ссылку (`vless://...`) в буфер iPhone
2. Открой Shadowrocket
3. Вкладка **Сервер** → плюс в правом верхнем углу
4. **Импортировать из буфера**
5. Профиль добавлен в список

**Способ 2 — QR-код:**
В 3x-ui панели у каждого клиента есть QR-код → сканируй его камерой Shadowrocket (плюс → сканировать QR).

## Правильные настройки Shadowrocket

Shadowrocket имеет много опций, часть из них ломает систему. **Включайте только из списка ниже.**

### Режим маршрутизации

Главный экран → кнопка **режима** (внизу, обычно написано «Настройка»).

| Режим | Когда использовать |
|---|---|
| **Прокси** (Global) | **В России** — весь трафик через VPN, без исключений. Самый стабильный вариант. |
| **Настройка** (Rule) | Если нет блокировок (например в Таиланде) — дефолтные правила. |
| **Прямой** | VPN отключен — НЕ использовать. |
| **Сцена** | Продвинутая фича со скриптами — не использовать. |

**Для РФ выбирай «Прокси».**

### Безопасные настройки (можно включать)

✅ **Тоннели → По требованию → По требованию** → ВКЛ
Автоматическое подключение когда нужен трафик.

✅ **Тоннели → По требованию → Всегда включено** → ВКЛ
**Kill Switch** — если VPN упал, трафик блокируется (защита от утечки реального IP).

✅ **Тоннели → По требованию → Показать сообщение об отключении** → ВКЛ
Уведомление в iOS если VPN отвалился.

✅ **Разрешения → Уведомления** → ВКЛ
Статус VPN в iOS уведомлениях.

✅ **Разрешения → Буфер обмена** → ВКЛ
Удобный импорт серверов через Ctrl+V.

✅ **Главный экран → Включить откат** → ВКЛ (если добавишь резервный сервер)
Автоматическое переключение на резерв при падении основного.

### Опасные настройки — НИКОГДА не включать

🔴 **Тоннели → Туннель → Включить все сети** (Force All Networks)
Эффект: Shadowrocket перехватывает трафик на всех сетевых интерфейсах включая внутренние iOS (iCloud Private Relay, AirDrop, Bonjour). Создаёт петлю или конфликт сертификатов.
Симптомы: SSL certificate verification failed, Shadowrocket вылетает, интернет пропадает, иногда крэш всего сетевого стэка.

🔴 **Общее → Модули** (любые модули)
Модули — это скрипты которые расшифровывают HTTPS для модификации трафика. Чтобы это работало, Shadowrocket подменяет SSL сертификаты на свой собственный CA. iOS видит «сертификат не от Apple/Google/банка» и **отказывается подключаться**.
Симптомы: SSL certificate verification failed на всех HTTPS-сайтах.

🔴 **Прокси → Цепь прокси**
Заворачивает трафик через 2+ серверов подряд. Если один из них падает — вся цепь падает.

🔴 **Прокси → Общий доступ к прокси**
Раздаёт твой VPN другим устройствам в локальной сети. В домашней Wi-Fi безопасно, в кафе/публичной сети — нет.

🟡 **Прокси → Режим совместимости**
Меняет способ перехвата трафика. Может вызывать неожиданное поведение. Не трогай без причины.

### TLS параметры (Тоннели → TCP)

Не трогай значения по умолчанию:
- `TLS: OpenSSL`
- `Отпечаток: Safari15_5`

Менять может сломать Reality handshake.

## Восстановление если что-то сломалось

**Универсальный план:**
1. **Настройки → Сброс → Сброс** — все настройки в дефолт
2. Серверы остаются, не пугайся
3. Подключись — проверь работу
4. Потом включай только настройки из списка «Безопасные», по одной, проверяя после каждой

## Для чего что используется

- **Telegram** — работает через любой прокси, если Shadowrocket включён
- **Claude Code через iOS Shortcuts / сторонние клиенты** — подхватит VPN прозрачно
- **Браузеры** — идут через прокси автоматически
- **Steam / игры** — обычно идут через прокси, регион определяется VPN-IP (Канада / Германия / что выбрал)

## Связка с iOS Shortcuts

Если хочешь автоматизацию: iOS Shortcuts может включать/выключать Shadowrocket по триггеру (геолокация, время, Wi-Fi сеть) через URL-схему `shadowrocket://`. Пример — выключать VPN дома, включать в кафе.

Документация URL-схем Shadowrocket: [Shadowrocket User Guide](http://shadowrocket.rocks/).
