# iOS Shadowrocket setup skill

Цель: провести пользователя через настройку Shadowrocket на iOS с импортом VLESS и правильными параметрами.

## Предусловия

1. **Shadowrocket куплен в US App Store**. $2.99. Если нет аккаунта США — направить на инструкцию установки аккаунта US App Store (это отдельный гайд)
2. **VLESS-ссылка** от 3x-ui
3. **iOS 14+**

## Ключевой tone для этого скилла

В отличие от Windows/Mac — нет автоматизации, только **шагно-по-шагово** с пользователем. Важно:
- Не перегружать всеми настройками сразу
- Сначала минимум чтобы заработало, потом опционально разрешения/уведомления/Kill Switch
- Явно предупреждать про опасные настройки (Force All Networks, модули)

## Пошагово для пользователя

### Шаг 1 — Импорт VLESS

Два способа:

**Через буфер:**
1. Пользователь копирует `vless://...` в буфер iPhone
2. Открывает Shadowrocket
3. Сервер → плюс → «Импортировать из буфера»

**QR-код:**
В 3x-ui есть QR для каждого клиента. Shadowrocket умеет сканировать (плюс → «Сканировать QR»).

### Шаг 2 — Режим «Прокси» (Global)

Главный экран → режим → **«Прокси»** (не «Настройка»/«Прямой»/«Сцена»).

Для РФ = весь трафик через VPN, самый стабильный вариант. Без headache с rule-set.

### Шаг 3 — Включить VPN

Переключатель на главном экране → ВКЛ. При первом включении iOS попросит добавить VPN-конфигурацию в Настройки → VPN. Подтвердить.

### Шаг 4 — Базовые настройки (опционально, рекомендуется)

- **Тоннели → По требованию → По требованию: ВКЛ** — автоподключение
- **Тоннели → По требованию → Всегда включено: ВКЛ** — Kill Switch
- **Разрешения → Уведомления: ВКЛ** — видеть статус VPN в iOS

### Шаг 5 — Проверка

Открой браузер, зайди на `ipinfo.io` или `api.ipify.org` — должен показать IP VPS, не реальный.

## Critical warnings (обязательно сказать пользователю)

**НИКОГДА не включать:**

1. **«Включить все сети» (Force All Networks)** в Тоннелях. Ломает iCloud Private Relay и крэшит сетевой стэк iOS. Инцидент 2026-04-11: рекомендация включить → крэш → пришлось делать полный сброс Shadowrocket.

2. **«Модули»** (Общее → Модули). Расшифровывают HTTPS → ломают SSL на банках, Apple, Google.

3. **«Цепь прокси»** (Прокси → Цепь). Ломается вся цепь если упадёт один сервер.

Если пользователь хочет что-то из этого включить — **явно объяснить риск** перед тем как разрешить.

## Error handling

**Симптом: «SSL certificate verification failed» в приложениях**
→ Включены модули или «Force All Networks». Отключить.

**Симптом: VPN включается, но сайты не открываются**
→ Режим «Прямой»? Проверить что выбран «Прокси» или «Настройка».

**Симптом: Shadowrocket вылетает при запуске**
→ Обычно после включения опасной настройки. Настройки → Сброс → Сброс. Серверы остаются.
